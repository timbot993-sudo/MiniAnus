# 🔐 Установка надежного пароля для админки

## Как это работает?

- Пароль **не хранится в коде** (безопаснее!)
- Проверка через **Netlify Function**
- Используется **SHA-256 хеширование**
- Пароль в **переменных окружения** (скрыт)

## Шаг 1: Выберите пароль

Примеры надежных паролей:

```
MyPortf0lio2024!Secret
SuperSecret#Admin123
P@ssw0rd!2024
```

## Шаг 2: Захешируйте пароль

### Вариант А: Онлайн (быстро)

1. Откройте https://www.sha256online.com/
2. Вставьте ваш пароль в поле
3. Скопируйте результат хеша (строка из 64 символов)

**Пример:**

```
Пароль: admin123
Хеш:    debe86d0d3f67a34bbd143e1285cd524f60f704f6f0a18d12ef35bd5e7e8f265
```

### Вариант Б: Node.js (если у вас установлен)

```bash
node -e "const crypto = require('crypto'); console.log(crypto.createHash('sha256').update('ваш_пароль').digest('hex'))"
```

## Шаг 3: Добавьте переменную в Netlify

1. Откройте **Netlify Dashboard**
2. Перейдите в **Site settings**
3. Выберите **Build & deploy** → **Environment**
4. Нажмите **Edit variables**
5. Добавьте новую переменную:
   ```
   Name:  ADMIN_PASSWORD_HASH
   Value: <ваш_хеш_из_шага_2>
   ```
6. Сохраните и пересоберите сайт

## Шаг 4: Протестируйте

1. Откройте `/admin/index.html` на сайте
2. Введите ваш пароль (не хеш, а оригинальный пароль!)
3. ✅ Если пароль верный - откроется админка

## ⚠️ Важно!

- **В поле пароля вводите оригинальный пароль**, не хеш!
- Пароль на Netlify синхронизируется только после пересборки сайта
- Не делитесь хешем пароля - это секретная информация!

## Что если забыли пароль?

1. Сгенерируйте новый пароль
2. Получите его хеш
3. Обновите `ADMIN_PASSWORD_HASH` в Netlify
4. Пересоберите сайт (`Trigger deploy`)

## Для локального тестирования

По умолчанию используется пароль **"admin123"** (хеш выше).
Для продакшена обязательно измените!
